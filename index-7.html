This is a Demo for CFQueryReader with ExtJS 5.0.0. ExtJS 5 is the latest version of ExtJS, and is a complete rewrite of the framework's data model. With that comes an entirely new CFQueryReader, that conforms to this new data model.


Adobe also released ColdFusion 11, which included the ability to add the queryFormat attribute to the ajax request, as well as a new queryFormat value of "struct". This example shows a request made with these parameters:




NOTE: CFQueryReader now supports both the "struct" queryFormat, as well as the ("row") default format. There are no plans to include support for the "column" queryFormat at this time.
