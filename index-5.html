This is a Demo for CFQueryReader with ExtJS 4.1.0. ExtJS 4 is the latest version of ExtJS, and is a complete rewrite of the entire framework. With that comes an entirely new CFQueryReader, that conforms to Ext's new class model. There is a single store defined in the script:

  • getWithMeta - The remote method returns a JSON serialized representation of a standard ColdFusion struct object. Within that struct is a metaData key, that defines the data. This can be used to change configuration of the store (or a grid, even) on the fly.
